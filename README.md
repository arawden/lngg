# LNGG
_(not to be confused with LNG, liquid natural gas)_

http://craftinginterpreters.com/introduction.html


## Running the Interpreter
From `.`:
- `javac jlox/*.java`
- `java jlox.Lox`

## Generating Syntax Tree Classes
From `jlox/`:
- `javac ./tool/GenerateAst.java`
- `java tool.GenerateAst .`
